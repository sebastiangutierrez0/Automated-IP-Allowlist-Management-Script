# Automated IP Allowlist Management Script

## Overview
This project automates IP allowlist maintenance by removing unauthorized or outdated IP addresses from a text file, reducing security risk caused by stale access permissions.

## Tech Stack
- Python

## Key Features
- Validates IP addresses against an approved list
- Removes unauthorized or outdated entries automatically
- Helps enforce least-privilege access through automation

## What I Learned
- Automating security hygiene tasks with Python
- Reducing manual access control errors
- Applying least-privilege principles in practice
